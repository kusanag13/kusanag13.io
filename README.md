# おまえシリーズクイズ

「おまえ〇〇なんだ、俺は〜だけど」という語呂合わせ系ネタを題材にしたひらがな入力クイズアプリです。

## 遊び方

1. 画面に「おまえ〇〇なんだ」という前半フレーズが表示されます
2. 「俺は \_\_\_\_ だけど」の空欄部分をひらがなで入力します
3. 「回答する」ボタン（または Enter キー）で答え合わせ
4. 正解・不正解を確認したら「次の問題へ」でランダムに次の問題へ進みます

スコア（正解数 / 不正解数 / 出題数）はセッション中に記録されます。
